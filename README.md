Game Arcade

Overview
Game Arcade is a browser-based web application developed using HTML, CSS, and JavaScript. It brings together multiple classic arcade games within a single platform, offering an interactive and engaging user experience. The project demonstrates fundamental front-end development concepts, including responsive web design, DOM manipulation, event handling, and game logic implementation.

Features
* Interactive Home Page
* User Login and Registration
* Password Recovery Page
* Rock Paper Scissors
* Tic Tac Toe
* Memory Card Game
* Leaderboard Interface
* Responsive and User-Friendly Design

Technologies Used
* HTML5
* CSS3
* JavaScript (ES6)
